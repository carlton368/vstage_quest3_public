=======================================

색상
----
- ColorMultiplier (색상 강도 계수)
  셰이더의 전체 밝기/강도를 조절합니다. 값이 높을수록 더 밝게 빛납니다.

패턴
----
- PatternSize (패턴 크기)
  점 또는 픽셀의 크기를 조절하여 홀로그램 패턴의 디테일을 조정합니다.

- Pattern (패턴 유형)
  드롭다운 메뉴에서 패턴 스타일을 선택합니다:
    * Dots (도트)
    * Pixels (픽셀)
    * (추가 스타일이 있다면 포함)

밉맵 기반 오라 (HologramUI.shader 전용)
--------------------------
*밉맵이 활성화된 텍스처가 필요합니다*

- MipLevel (밉맵 레벨)
  오라 샘플링에 사용할 밉맵 레벨을 정의합니다. 값이 클수록 부드러운 가장자리 생성.

- MipLevelIntensity (밉맵 강도)
  밉맵 기반 프레넬 효과의 강도를 조절합니다. 0은 비활성화, 값이 클수록 강해집니다.

글리치
------
- GlitchAmount (글리치 양)
  홀로그램에 적용되는 공간적/시간적 글리치 양을 조절합니다.

- GlitchMinOpacity (최소 불투명도)
  글리치 효과 중 적용될 최소 불투명도입니다. 낮은 값일수록 더 투명해집니다.

- GlitchTiling (글리치 타일링)
  글리치 텍스처의 타일링 밀도를 조절합니다.

- DistortionAmount (왜곡 강도)
  스크린 공간에서 UV 왜곡의 강도를 조절합니다. 값이 클수록 더 심한 흔들림 효과.

텍스트 셰이더 전용 설정
------------------------
- TextDilate (텍스트 확장)
  텍스트의 굵기를 늘립니다. 강조 효과나 발광 표현에 사용됩니다.

- TextDilateExtension (확장 테두리)
  외곽을 부드럽게 확장하여 빛 번짐 효과를 만듭니다.

렌더 설정
---------
- Render Queue (렌더 큐)
  다른 객체보다 먼저 또는 나중에 렌더할지 순서를 지정합니다. 기본값은 3000 (투명 객체).
  일반적으로 "From Shader"로 유지 권장.

- GPU 인스턴싱 활성화
  Unity가 동일 머티리얼 인스턴스를 배치 처리할 수 있도록 합니다.

- 양면 글로벌 조명
  UI 양면에 빛을 적용합니다. Light Probe나 GI 사용할 경우 유용합니다.

- 글로벌 조명
  해당 머티리얼이 글로벌 조명에 어떻게 참여할지 설정합니다. 일반적으로 None으로 설정.
